# 💕 Happy 1st Anniversary Website

A beautiful, romantic anniversary website created with love for your special someone.

## ✨ Features

- **Romantic Hero Section** - Beautiful animated hearts and heartfelt message
- **Countdown Timer** - Shows how long you've been together
- **Love Timeline** - Your relationship milestones beautifully displayed
- **Photo Gallery** - Space for your precious memories together
- **Love Letter** - A heartfelt message section
- **Beautiful Animations** - Floating hearts, smooth transitions, and romantic effects
- **Responsive Design** - Looks perfect on all devices

## 🎨 Customization

### Change the Anniversary Date
Edit `src/App.jsx` and modify this line:
```javascript
const anniversaryDate = new Date('2025-05-28')
```
Change the date to your actual anniversary.

### Customize the Love Letter
Edit the letter content in the "Love Letter Section" of `src/App.jsx` to make it personal and special.

### Add Your Photos
Replace the placeholder photo cards in the gallery section with your actual photos. You can:
1. Add your photos to a `public/photos` folder
2. Update the `photos` array in `src/App.jsx` with your image paths

### Customize Memories
Update the `memories` array in `src/App.jsx` with your actual relationship milestones.

## 🚀 Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

- **Lucide React** - Beautiful icons
